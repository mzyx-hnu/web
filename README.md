# HTML 战棋模拟器开发仓库

欢迎来到这个 HTML 战棋模拟器仓库！这里包含一个基于 HTML5, CSS (Tailwind) 和 JavaScript 开发的回合制战棋游戏。

## 项目介绍

### 10×10 战棋模拟器 (火焰纹章 Lite) - `index.html`
一个经典的 10x10 网格回合制战棋游戏。
- **地形系统**：山峰（不可通行）、河流（限制移动）、森林（防御/范围修正）。
- **战斗系统**：多种职业（骑士、弓箭手、法师等），拥有独立的属性、普通技能与终极技能。
- **编辑器模式**：内置单位编辑器，可以直接在棋盘上新增、修改或复制单位属性。
- **存档功能**：支持将当前棋盘状态导出为 YAML 文件，并可随时重新导入。

## 如何运行

本游戏是一个单文件的 HTML 应用程序（或重构后的多文件结构），运行非常简单：
1. 下载或克隆本仓库。
2. 在电脑上找到 `index.html`。
3. 双击文件，使用任何现代浏览器（Chrome, Edge, Firefox, Safari 等）打开即可开始游戏。

## 开发者说明

项目经过重构，实现了样式、逻辑与结构的分离，更便于扩展：
- **结构 (HTML)**：`index.html` 定义了游戏的 UI 布局。
- **样式 (CSS)**：`style.css` 负责游戏的视觉表现，结合了 Tailwind CSS。
- **逻辑 (JS)**：`script.js` 驱动所有的游戏规则、动画与交互逻辑。
