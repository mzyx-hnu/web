# HTML 小游戏开发仓库

欢迎来到这个 HTML 小游戏仓库！这里包含两个基于 HTML5, CSS (Tailwind) 和 JavaScript 开发的有趣小游戏。

## 项目列表

### 1. 月抛模拟器 (Panic Edition) - `1.html`
一个关于约会风险与决策的选择类模拟游戏。
- **核心机制**：生理压抑 (Urge) 与心理压力 (Anxiety) 的平衡。
- **延迟判决**：高危行为后不会立即得知结果，只能通过去医院检查或最终结算揭晓。
- **视觉反馈**：高压力下会出现视线模糊等特效。
- **系统功能**：包含道具系统（试纸）、医院检查、详细的约会记录复盘。

### 2. 10×10 战棋模拟器 (火焰纹章 Lite) - `2.html`
一个经典的 10x10 网格回合制战棋游戏。
- **地形系统**：山峰（不可通行）、河流（限制移动）、森林（防御/范围修正）。
- **战斗系统**：多种职业（骑士、弓箭手、法师等），拥有独立的属性、普通技能与终极技能。
- **编辑器模式**：内置单位编辑器，可以直接在棋盘上新增、修改或复制单位属性。
- **存档功能**：支持将当前棋盘状态导出为 YAML 文件，并可随时重新导入。

## 如何运行

这些游戏都是单文件的 HTML 应用程序，不需要复杂的安装步骤：
1. 下载或克隆本仓库。
2. 在电脑上找到 `1.html` 或 `2.html`。
3. 双击文件，使用任何现代浏览器（Chrome, Edge, Firefox, Safari 等）打开即可开始游戏。

## 开发者说明

由于项目采用单文件结构（HTML/CSS/JS 合一），非常适合快速原型开发。
- **样式**：使用了 Tailwind CSS CDN。
- **数据**：`2.html` 使用了 JS-YAML 处理棋盘数据。
- **交互**：纯原生 JavaScript 驱动。
